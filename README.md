# Mechanical Design Projects

A collection of mechanical design and structural optimization study projects focused on **geometry-performance tradeoffs** in engineering components.

This repository currently contains two project reports:

1. **Optimal design of a railway rail web cross-section** considering cooling efficiency and critical buckling load
2. **Stress concentration reduction in a double-nut system** through taper-angle optimization

The purpose of this repository is to document small but focused mechanical design projects that combine **engineering reasoning**, **design variables**, and **performance evaluation**.

---

## Overview

This repository is not intended as a software-heavy codebase.  
Instead, it serves as a compact portfolio of **mechanical design problem-solving projects**, where the main outputs are engineering analyses, optimization logic, and final design reports.

The common theme across the projects is:

- defining a mechanical performance objective
- selecting a geometric design variable
- evaluating the tradeoff between competing criteria
- proposing an improved design direction

## Included Projects

### 1. Optimal Design of a Railway Rail Web Cross-Section
**File:** `냉각 효율과 임계 좌굴 하중을 고려한 철도 레일 웹(Web) 단면의 최적 설계.pdf`

This project studies the cross-sectional design of a railway rail web while balancing:

- **cooling efficiency**
- **critical buckling load**

The design problem is treated as a tradeoff between thermal performance and structural stability, with the goal of identifying a cross-sectional configuration that improves cooling behavior without excessively sacrificing load-bearing safety.

### 2. Stress Concentration Reduction in a Double-Nut System
**File:** `이중 너트 시스템의 테이퍼 각도 최적화를 통한 응력 집중 완화.pdf`

This project investigates how the taper angle in a double-nut system affects:

- **stress concentration**
- **load distribution**
- **structural reliability**

The central idea is to reduce localized peak stress by adjusting geometric features rather than changing the entire fastening concept.
